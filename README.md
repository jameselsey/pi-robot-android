pi-robot-android
=========

pi-robot-android is an android client for the [BrickPi robot server][1]

How to use
----

  - clone this project
  - import into intelliJ
  - Build and deploy to device
  - (ensure IP/ports set correctly)

Requirements
----
Code is tested on the following, your results may vary.

  - IntelliJ 13.1.1
  - Gradle 1.11
